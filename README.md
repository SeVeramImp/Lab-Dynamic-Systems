# Lab-Dynamic-Systems
Репозиторий содержит материалы лабораторной работы по исследованию дискретных динамических систем на примере логистического отображения
и его модификаций.

## Содержание

1. lab.pdf — отчёт по проделанной работе

2. src/ — каталог с исходниками
   - lab.tex — tex-файл отчёта
   - reference.pdf — техзадание для лабы
   - code/
     - easy.py — код для Easy-части
     - normal.py — код для Normal-части
     - hard.py — код для Hard-части
   - images/ — каталог для хранения графиков и изображений
